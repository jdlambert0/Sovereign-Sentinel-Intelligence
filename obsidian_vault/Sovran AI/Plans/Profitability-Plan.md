# Profitability Plan (Phase 8)

Goal
- Achieve sustainable profitability for Sovran AI with atomic bracket integration (Phase 7) and full end-to-end automation.

Strategic Plan
- Phase 1-2: Stabilize atomic bracket automation; ensure robust tests and logging.
- Phase 3-4: Incremental automation on multi-symbol trading; implement robust risk controls and monitoring.
- Phase 5: Achieve a target monthly recurring net profit (e.g., $3k-5k) with healthcare security in mind.

Key Metrics
- Win rate, average P/L per trade, max drawdown, days to profitability, and risk-adjusted returns (Sharpe/Sortino).
- Daily/Monthly Revenue tracking: Emoify vs bFriend lines in long term context.

Risk Budget & Controls
- Cap daily loss to a defined threshold; escalate if breached.
- Implement memory/logging for compliance/auditing.
- Phase exit plan if risk tolerance is breached.

Phased Roadmap
- Phase 8.1 (0-60 days): finalize atomic bracket integration in production; ensure monitoring and rollback.
- Phase 8.2 (60-120 days): expand multi-symbol trading with stable risk controls; test extensively.
- Phase 8.3 (120-180 days): scale operations and pursue enterprise-level pilots.
